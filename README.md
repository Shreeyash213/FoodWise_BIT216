# SavePlate_BIT216
SavePlate is a web application designed to reduce household food waste 
through inventory management, expiry alerts, donation facilitation, 
and meal planning.

## Features
- User Registration & Privacy Settings
- Food Inventory Management
- Browse & Donate Food Items
- Food Analytics & Impact Tracking
- Notifications
- Weekly Meal Planning

## Tech Stack
PHP | MySQL | Bootstrap 5 | HTML | CSS

## Setup Instructions
1. Install XAMPP and start Apache + MySQL
2. Import `database/saveplate.sql` into phpMyAdmin
3. Configure DB credentials in `src/config.php`
4. Open `localhost/SavePlate_BIT216/src/` in browser
